# Autopsy Cheat Sheet - Digital Forensics

### Core Workflow
1. Create New Case
2. Add Data Source (Disk Image / Folder / Logical Files)
3. Configure Ingest Modules
4. Wait for analysis to complete
5. Review results using different views

### Key Views & Modules
- **Timeline** — Chronological events
- **Keyword Search** — Custom keyword lists
- **File Analysis** — Deleted / carved files
- **Web Artifacts** — Browser history, cookies
- **Email** — Parse email clients
- **Hash Lookup** — Known good/bad files

**Pro Tips**:
- Enable all relevant ingest modules
- Use "Interesting Files" module
- Export reports for documentation
- Maintain chain of custody in real cases
