# Wazuh Cheat Sheet - Open Source SIEM

### Key Capabilities
- Log Analysis & Correlation
- File Integrity Monitoring (FIM)
- Vulnerability Detection
- Active Response (block IPs, etc.)
- Cloud security monitoring
- Container security

### Common Agents Deployment
- Linux: Install agent package
- Windows: MSI installer
- Register agent with manager

**Dashboard Highlights**:
- Security Events
- Integrity Monitoring
- Vulnerabilities
- Compliance (PCI-DSS, GDPR, etc.)

**Pro Tips**:
- Tune rules to reduce false positives
- Use custom decoders and rules
- Set up email/Slack alerts
- Monitor critical assets first
